  • A cut-off that is generally accepted (but should be reported)

  2. We can discard questions that dont meet this criteria

Step 2. EFA: Parallel analysis (cont.)

  1. artificial datasets is generated with the same number of variables and same number of observations
  2. Variables in these datasets are random variables
  3. Factor analysis is performed on each of these datasets and the eigenvalues of each factor generated are extracted
  4. Repeating this many times will allow us to calculate the mean eigenvalue of a factor from datasets similar to what we have
  5. An eigenvalue less than the mean would mean that a particular factor extracted is no more substantial (explains less) than a random factor (which really has no meaning); those factors are rejected
